# Content  
- Git
	- List all branch ```git branch -a```
	- Revert commit ```git reset HEAD^```
	- Checkout to remote branch with detached HEAD ```git checkout -t branch_name```
	- Log ```git log --graph --pretty=oneline --abbrev-commit```
